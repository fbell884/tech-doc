Declaring Variables in JavaScript

- The first line in this code is declaring a variable named: "myName".

- The second line in this code is assigning a string value of "Francis" to "myName.

var myName;
myName = "Francis";

    • Numbers in JavaScript

    - The first line of this code is declaring a number variable and assigning it a value.

    - The second and fourth lines of code provide the result of adding 1 to the "myNumber" variable.

    var myNumber = 10;
    myNumber = myNumber + 1;
    // The "++" operator provides the same result as the code above
    myNumber++;

    • If-Else Statements in JavaScript

    - Here is an example of an If-Statement in JavaScript

    - In this code, there is a "num" variable. If this number is great than 15, the code returns the string: "Bigger than 15"
    - If the number is less than 5, it returns the string "Smaller than 5"
    - If the number does not match these two conditions (if it is in between 5 and 15), it returns "Between 5 and 15"

    if (num > 15)
    {
    return "Bigger than 15";
    }
    else if (num < 5)
    {
    return "Smaller than 5";
    }
    else {
    return "Between 5 and 15";
    }
